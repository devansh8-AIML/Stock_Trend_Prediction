# 📈 Stock Trend Prediction using Hybrid Deep Learning Models

A deep learning-based stock market prediction system that forecasts stock price trends using historical market data. The project uses **BiLSTM** and **GRU** neural network architectures to capture temporal dependencies and improve prediction accuracy.

This project demonstrates the use of Artificial Intelligence and Time-Series Forecasting in financial market analysis.

---

## 🚀 Features

- Predicts future stock trends using historical data
- Uses Hybrid Deep Learning Models (BiLSTM + GRU)
- Real-time stock data fetched using Yahoo Finance API
- Data preprocessing and feature scaling
- Trend visualization with graphs
- Performance evaluation using metrics
